# ⚽ Python Football CRUD Project

## 📌 Overview

A simple **CRUD application** built using **Core Python + SQLite**.
This project demonstrates backend fundamentals similar to Laravel (MVC-like structure).

---

## 🚀 Features

* ➕ Add Player
* 📄 View Players
* ✏️ Update Player
* ❌ Delete Player
* 🔍 Search Player
* 📊 Table view using `tabulate`
* 🔐 Input validation

---

## 🛠️ Tech Stack

* Python 3
* SQLite (built-in database)
* Tabulate

---

## 📁 Project Structure

```
main.py        → Entry point  
controller.py  → Business logic  
player.py      → Model  
db.py          → Database connection  
```

---

## ▶️ How to Run

```bash
pip install tabulate
python main.py
```
